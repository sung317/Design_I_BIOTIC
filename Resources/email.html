To follow up on our meeting today, I wanted to quickly provide a bit of literature to help put the program in context and describe some of the algorithms on the back end.  This may or may not be helpful for your upcoming assignments/presentations.


- Chang et.al. is a review of Quantitative MRI (qMRI) methods focusing on T1 and T2 mapping, which both gives some basic physics/equations describing the relaxation processes, and provides some examples of where these techniques might be useful clinically.

- Barral et.al. describes the linearized T1 fitting algorithm that you'll see implemented in the T1 mapping code I will send along shortly.  None of this part should have to be modified during this project but I've included it for your reference.

- Pei et.al. describes ARLO, which is a linearized T2 fitting algorithm that I'd like to try and implement eventually - this would replace the core of the current T2 fitting algorithm which uses a much slower non-linear least-squares fit.


While it should not be immediately necessary for the project I'd be happy to provide some more basic MRI literature if you are interested - Steven teaches a medical physics course that is ~1/3 MRI and almost certainly has some slides that would be a good starting point.
